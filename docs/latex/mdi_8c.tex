Determine the conversion factor between two units. 

The function determines the conversion factor from {\ttfamily in\+\_\+unit} to {\ttfamily out\+\_\+unit}. The function requires that {\ttfamily in\+\_\+unit} and {\ttfamily out\+\_\+unit} be members of the same category of unit ({\itshape i.\+e}. charge, energy, force, etc.). For example, calling {\ttfamily M\+D\+I\+\_\+\+Conversion\+\_\+\+Factor}({\ttfamily \char`\"{}kilojoule\+\_\+per\+\_\+mol\char`\"{}},{\ttfamily \char`\"{}atomic\+\_\+unit\+\_\+of\+\_\+energy\char`\"{}}) will return the conversion factor from kilojoule/mol to hartrees. The function returns {\ttfamily 0} on a success.

All quantities communicated through M\+DI must be represented using atomic units. When unit conversions are necessary, this function should be used to obtain the conversion factors, as this will ensure that all drivers and engines use conversion factors that are self-\/consistent across codes. Use of conversion factors that are not self-\/consistent can result in numerical instabilities.
